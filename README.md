# Growth Lab

Jac fullstack app for auditing consumer app retention loops, screenshot-based UX/design psychology, and manipulative engagement risk.

## Run locally

```powershell
jac start --dev main.jac
```

The app uses Jac server endpoints plus client-side Jac components.
Audits capture sanitized HTML evidence plus safe headless-browser screenshots and same-origin click-through evidence. Forms, purchases, credential entry, and destructive actions are avoided.
Recommendations focus on broad retention science: cues, actions, rewards, learning loops, stopping cues, notifications, streak pressure, social validation, FoMO, personalization controls, and dark-pattern risk.
